CSS GRID Basics

Maria Larchenko

CSS Grid is two dimentional layout system. Unlike flexbox, which is largerly a 1-dimentional system, CSS grid can handle both row and column.

Grid is support in the latest version of all popular browser

Basic concepts

  • Grid-container
  • Grid-item
  • Grid-gap
  • Grid-template-rows
  • Grid-template-column
  • Grid-template-areas
Create a simple page with div like grid conteiner. Inside the container there are 4 items. 
	
	  
		

		  
 Header 

		  
 Main 

		  
 Sidebar 

		  
 Footer

To start using Grid you have to define property display: grid. Display property declaring on the container element. 

			  
					 
      .wrapper {
        display: grid;

      }

Next define rows and columns. You can do it using the property grid-template-columns and grid-template-rows. 

				
		.wrapper {
			display: grid;
			grid-template-columns: 1000px 500px;
			grid-template-rows: 200px 200px 200px;
		}
		
	

		  
				 
    .wrapper {
      display: grid;
      grid-template-columns:  2fr 1fr;
	    grid-template-rows: 1fr 1fr 1fr;
	    grid-gap: 20px;

    }

		  
		
			  
					 
	  .wrapper {
	    display: grid;
	    grid-template-columns: 2fr 1fr;
	    grid-template-rows: repeat(3, 1fr);
	    grid-gap: 20px;
	  }
	
			  
			
				  
						 
		.wrapper {
		  display: grid;
		  grid-template-columns: minmax(600px, 2fr) 1fr;
		  grid-template-rows: repeat(3, 1fr);
		  grid-gap: 20px;
		}

Position Items

  • Grid-row-start
  • Grid-row-end
  • Grid-column-start
  • Grid-column-end
  • Grid-row
  • Grid-column
  • Grid-areas
  • Align-self
  • Justify-self
					  			 
      .footer {
        grid-row: 3;
        grid-column: 1/-1;
      }
        
      .main {
        grid-column: 1;
        grid-row: 2;
      }
        
      .header {
        grid-column: 1/-1;
        grid-row: 1;
      }
        
      .aside {
        grid-column:2;
        grid-row:2;
      }

Grid-template-areas

							  
									 
      .wrapper {
        display: grid;
        grid-template-columns: 2fr 1fr;
        grid-template-rows: repeat(3, 1fr);
        grid-template-areas: "header  header"
        						         "main  aside"
        						         "footer footer";
      }

        
      .footer {
        grid-area: footer;
      }
        
      .main {
        grid-area: main;
      }
        
      .header {
        grid-area: header;
      }
        
      .aside {
        grid-area: aside;
      }

Justify-self

Aligns a grid item inside a cell along the row axis

Align-self

Aligns a grid item inside a cell along the column axis
